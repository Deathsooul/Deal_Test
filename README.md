# Deal Interview Test

## Requirements
- [**Git**](https://git-scm.com/) para clonar o projeto.
- [**Node.js**](https://nodejs.org/en/) instalado.
- [**Docker**](https://www.docker.com/)

### Running Code
* Dev Mode
```
yarn start
```

* Dev Mode
```
yarn dev
```


# Running Docker

### Start
```
docker-compose up
```

### Stop
```
docker-compose down
```
